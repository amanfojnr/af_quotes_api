Access api to generate random quote via

https://af-quotes-api.herokuapp.com/api/quotes/random/

JSON response sample

    { 
        "id":     160,
        "quote":  "There are many colorful flowers on the path of life, 
                   but the prettiest have the sharpest thorns. ",
        "source": "African Proverb"
    }

Access api to generate a number of quotes using a limit parameter via

https://af-quotes-api.herokuapp.com/api/quotes/list/<limit>
Where limit is an integer. if limit exceeds the number of quotes in the database, the api responds with all quotes.

You can also get a list of all quotes in the database via

https://af-quotes-api.herokuapp.com/api/quotes/list
